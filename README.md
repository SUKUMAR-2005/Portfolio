# Sukumar M — Portfolio Website

A personal portfolio website built with pure HTML, CSS, and JavaScript.

🔗 **Live Site:** [https://sukumar-dev.vercel.app](https://sukumar-dev.vercel.app/)

---

## 🚀 Features

- Responsive design — works on mobile, tablet, and desktop
- Smooth scroll reveal animations
- Professional custom cursor
- Sticky navbar with active section highlight
- Working contact form (via Formspree)
- Dark theme with accent colors

## 📁 Project Structure

```
sukumar-portfolio/
├── index.html        # Main portfolio file (all-in-one)
├── README.md         # This file
└── .nojekyll         # Tells GitHub Pages to skip Jekyll processing
```

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts — Plus Jakarta Sans, JetBrains Mono, Fraunces
- Formspree (contact form backend)



## 🌐 Deploying to GitHub Pages

1. Push this folder to a GitHub repository named `sukumar-portfolio`
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save** — your site will be live in ~1 minute!
